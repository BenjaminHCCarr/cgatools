const char *  last 
) [inline]

Parse the string to the given Value type.

Calling parseValue<Value>(str) is much like calling boost::lexical_cast<Value>(str), with the following exceptions:

  • If Value is an 8-bit integer, boost may treat this as a string to character conversion, while parseValue still treats it as a string to integer conversion.
  • Integer conversion by parseValue is very strict about only finding digits, possibly starting with a '-' character.
  • This is much faster than boost::lexical_cast.
  • This is only implemented for the following types:
    • cgatools::int8_t
    • cgatools::int16_t
    • cgatools::int32_t
    • cgatools::int64_t
    • cgatools::uint8_t
    • cgatools::uint16_t
    • cgatools::uint32_t
    • cgatools::uint64_t
    • float
    • double

IntType *  pVal 
) [inline]

Reads an integer stored in "zero-compressed" format.

At each iteration, a byte is read in, and the lsb 7 bits are or'd into val. If the msb is set, then the val is shifted left by 7, and the next byte must be read in.
