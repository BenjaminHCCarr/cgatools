Detailed Description

This file contains definitions of classes used to manipulate junctions and files containing them.

A Junction describes a section of assembled sequence consisting of:

  • A left section which maps, exactly or approximately, to a section of the reference genome (either strand).
  • A transition section..
  • A right section which maps, exactly or approximately, to a section of the reference genome (either strand).
