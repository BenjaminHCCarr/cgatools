cgatools 1.8.0

The Complete Genomics Analysis Tools (cgatools) is an open source project to provide tools for downstream analysis of Complete Genomics data. The focus is to provide command line utilities. A C++ API is available as well. For detailed install instructions, as well as motivation, design decisions, general usage of the tools, and interpretation of the results, please see the user guide.

The general areas of functionality include:

  • Genome comparison
    • snpdiff
    • calldiff
    • listvariants (beta)
    • testvariants (beta)
    • junctiondiff (beta)
    • mkvcf (beta)
  • Format conversion and File Manipulation
    • generatemastervar (beta)
    • mkvcf (beta)
    • varfilter (beta)
    • evidence2sam (beta)
  • Annotation
    • join (beta)
    • junctions2events (beta)
  • Reference tools
    • fasta2crr
    • crr2fasta
    • listcrr
    • decodecrr
